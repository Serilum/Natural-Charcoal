<h2>Mod Description</h2>
<p style="text-align:center"><a href="https://serilum.com/" rel="nofollow"><img src="https://cdn.serilum.com/description/header/header.png" alt="" width="838" height="400"></a></p>
<p style="text-align:center"><a href="https://curseforge.com/members/serilum/projects" target="_blank" rel="nofollow"><img src="https://workflow.serilum.com/badges/svg/curseforge.svg" width="200"></a> <a href="https://modrinth.com/user/Serilum" target="_blank" rel="nofollow"><img src="https://workflow.serilum.com/badges/svg/modrinth.svg" width="200"></a> <a href="https://patreon.com/serilum" target="_blank" rel="nofollow"><img src="https://workflow.serilum.com/badges/svg/patreon.svg" width="200"></a> <a href="https://youtube.com/@serilum" target="_blank" rel="nofollow"><img src="https://workflow.serilum.com/badges/svg/youtube.svg" width="200"></a></p>
<p><strong><span style="font-size:24px">Requires the library mod&nbsp;<a style="font-size:24px" href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow">Collective</a>.<br></span></strong></p>
<p><span style="font-size:18px">Natural Charcoal makes wildfires useful! Whenever a log is destroyed via fire, it has a chance to drop charcoal. Charcoal is immune to fire, so it doesn't get destroyed by other fires nearby. By default it's a 2/3 chance for one to generate, though this can be changed in the config.<br><br>It is also possible to put logs next to a campfire to turn them into charcoal. This is slightly faster than using a furnace: Four pieces in 30 seconds instead of 40 when smelted.<br></span><br><br><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://github.com/Serilum/.information/wiki/how-to-configure-mods" target="_blank" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span><br></strong><span style="font-size:12px"><strong>charcoalIsImmuneToFire</strong>&nbsp;(default = true): Whether the charcoal item should survive when being dropped inside fire.</span><br><span style="font-size:12px"><strong>burnedLogBecomesCharcoalChance</strong>&nbsp;(default = 0.67, min 0, max 1.0): The chance a log block that has been burned turns into charcoal.</span><br><span style="font-size:12px"><strong>charcoalDropAmount</strong>&nbsp;(default = 1, min 1, max 64): How much charcoal should be dropped from a burned log.</span><br><br><br><span style="font-size:18px"><strong>A fire passed through the area! Useful, as it generates charcoal :)</strong></span><br><picture><img src="https://cdn.serilum.com/projects/natural-charcoal/a.png"></picture><br><br><span style="font-size:18px"><strong>Charcoal has a chance to drop whenever a log is destroyed via fire:</strong></span></p>
